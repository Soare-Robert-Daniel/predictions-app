## Simple Prediction App

This is a simple prediction app similar to Polymarket or Khalsy, where a user can create a prediction market, and other users can buy shares in the market.

Unlike a traditional prediction market, this will use votes instead for shares. Each user can only vote once, and the outcome of the market will be determined by the majority vote.

### User Stories

1. As a user, I want to create a prediction market with a question and multiple options (e.g., Yes/No, Option A/Option B, etc.).
2. As a user, I want to vote on a prediction market by selecting one of the options.
3. As a user, I want to see the current votes for each option in the prediction market.
4. As a user, I want to see the outcome of the prediction market once the voting period has ended.
5. As a user, I want to receive a notification when the outcome of the prediction market is determined.

### Admin Stories

The admin can create a prediction market, set the voting period, and determine the outcome of the market once the voting period has ended. The admin can not vote in the market.
